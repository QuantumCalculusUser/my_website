# Portfolio Website

A modern, responsive portfolio website showcasing three interactive web applications built with HTML, CSS, and JavaScript.

## Features

### Projects Included

1. **E-Commerce Website** - A fully functional shopping cart application
   - Product listing with images
   - Add/remove items from cart
   - Real-time price calculation
   - Responsive product grid

2. **Task Management App** - An interactive task organization tool
   - Add, complete, and delete tasks
   - Priority levels (Low, Medium, High)
   - Filter tasks by status (All, Pending, Completed)
   - Local storage persistence

3. **Weather Dashboard** - A real-time weather application
   - Search weather by city name
   - Current weather display with dynamic color themes
   - 5-day weather forecast
   - Temperature-based visual styling
   - Responsive design with smooth transitions

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, Custom Properties, Transitions
- **JavaScript (ES6+)** - Async/await, Event handling, DOM manipulation
- **APIs** - Open-Meteo Geocoding and Weather APIs
- **Storage** - Local Storage for task persistence

## Setup & Usage

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Click on any project to explore its features
